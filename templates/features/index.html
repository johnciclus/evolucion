Lenguaje en prosa.

El lenguaje en prosa permite describir el fenómeno de manera textual, en Evolución web se integra un editor de texto enriquecido que definir este lenguaje de la dinámica de sistemas.

Lenguaje de influencias.

El lenguaje de influencia describe mediante elementos, ciclos y relaciones visuales, una visión general del fenómeno permitiendo identificar los ciclos de realimentación.

Lenguaje de flujo nivel.

El lenguaje de flujo nivel permite modelar la estructura del fenómeno en estudio mediante un lenguaje visual que tiene asociado ecuaciones matemáticas, las cuales permiten la simulación del sistema.

Lenguaje de Ecuaciones.

El lenguaje de Ecuaciones representa la estructura del fenómeno mediante un conjunto de ecuaciones diferenciales que permiten simular el comportamento del sistema.

Lenguaje de Comportamientos.

El lenguaje de Comportamientos permite analizar visualmente el comportamiento del sistema con la finalidad de validar si simula aproximadamente el sistema.
